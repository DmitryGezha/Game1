Игра-головоломка от 1-го лица.
Вдохновлялся такими играми, как Portal, Q.U.B.E., Antichamber.
Цель - идти вперед, решать головоломки, переходить из комнаты в комнату.
Управление стандартное - можно ходить, приседать, прыгать, брать предметы.
Основная особенность игры - предметы не существуют без наблюдателя.
Внутри папки - подсказки и прохождение. Удачи.
